# Ghost Thermostat Positioning One-Pager

Last updated: 2026-04-28

## Elevator Pitch

Ghost Thermostat is a simpler control design built for operational reliability: faster to deploy, easier to validate, and safer to recover when faults occur.

## The Problem

Many thermostat/control solutions add complexity that increases install variability, troubleshooting time, and operational risk.

## Our Approach

Ghost Thermostat prioritizes:

- Minimal, clear control logic
- Repeatable commissioning steps
- Explicit fail-safe behavior
- Evidence-backed validation standards

## Why It Wins

- **Faster deployment:** less ambiguity in setup and commissioning
- **Lower operational burden:** predictable behavior across standard scenarios
- **Stronger reliability posture:** documented response to faults and restart events
- **Scalable delivery:** process can be repeated by teams, not just individuals

## Proof Framework

Every release is validated through three core scenarios:

1. Normal cycle stability
2. Sensor fault handling
3. Power loss and recovery

Each test run captures measurable results, pass/fail outcomes, and corrective actions.

## Ideal Users

- BuilderOps teams managing installations at scale
- Operators needing clear maintenance and recovery procedures
- Stakeholders who value predictable performance over feature bloat

## Implementation Readiness

This project includes:

- Operational runbook
- Style and process standards
- Contribution and change-control workflow
- Traceable changelog and governance artifacts

## Bottom Line

Ghost Thermostat turns a unique simple design into a repeatable, professional operating system for real-world delivery.
