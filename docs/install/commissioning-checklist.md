# Commissioning Checklist

Last updated: 2026-04-28

Use this checklist to standardize installation verification and avoid ad-hoc commissioning.

## 1) Pre-Install Verification

- [ ] Client intake form completed and reviewed.
- [ ] Site access and timing confirmed.
- [ ] Existing HVAC type and stages verified.
- [ ] Required tools and replacement components available.
- [ ] Baseline photos and wiring labels captured.

## 2) Wiring and Physical Installation

- [ ] Power isolated before wiring changes.
- [ ] Wire labels matched to expected terminals.
- [ ] Common wire and power requirements verified.
- [ ] Mounting location appropriate for representative temperature sensing.
- [ ] No loose conductors or exposed contacts.

## 3) Initial Power-On and Safety Checks

- [ ] Safe startup sequence completed.
- [ ] No immediate fault indicators present.
- [ ] Relay behavior aligns with commanded state.
- [ ] Minimum cycle protection configured.
- [ ] Default safe-state behavior verified.

## 4) Control Validation

- [ ] Heating call test passed.
- [ ] Cooling call test passed (if applicable).
- [ ] Setpoint approach and deadband behavior verified.
- [ ] Manual override and return-to-auto behavior verified.
- [ ] Runtime logging active and readable.

## 5) Fault and Recovery Verification

- [ ] Sensor fault handling spot-checked.
- [ ] Power interruption recovery verified.
- [ ] Fault visibility and operator guidance confirmed.
- [ ] No unresolved high-severity anomalies.

## 6) Handover and Sign-off

- [ ] Operator briefed on normal operation and override policy.
- [ ] Operator briefed on fault response path.
- [ ] Pilot success metrics reviewed with stakeholder.
- [ ] Weekly status reporting cadence agreed.
- [ ] Commissioning sign-off completed.
